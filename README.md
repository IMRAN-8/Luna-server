# Luna AI

Luna AI is an ESP32-based AI voice assistant powered by a Node.js backend.  
The ESP32 records the user's voice and sends it into text, think using AI, remember useful information, and reply back with voice.

The goal of this project is to connect **IoT hardware** with **AI services** and create a simple smart assistant that can run from an ESP32 device.

---

## Features
```
•	Voice interaction using ESP32
•	Speech-to-text processing
•	AI-powered chat response
•	Text-to-speech voice reply
•	Memory system for storing useful facts
•	Web server backend using Node.js
•	Simple browser page support
•	Designed for IoT and embedded AI experiments

```

## How It Works

```txt
User Voice
    ↓
ESP32 Microphone
   ↓
Luna AI Server
   ↓
Speech-to-Text
   ↓
AI Chat Processing
   ↓
Text-to-Speech
   ↓
ESP32 Speaker

```

## Tech Stack
```
•	Node.js
•	Express.js
•	JavaScript
•	ESP32
•	I2S Microphone
•	I2S Speaker Amplifier
•	OLED Display
•	Cloud-hosted backend
```

Install dependencies:

```
npm install
Run the server:
npm start
or:
node server.js
```
